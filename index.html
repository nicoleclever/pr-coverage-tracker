PR Coverage Tracker

How to use: Studies are automatically pulled from the master link sheet. Set your start date in the From field, then click Start Tracking to pull all coverage links since that date. Use the site dropdown to filter by domain, or use the search box to find results by keyword.

The Copy all for spreadsheet button will copy the data in the exact column order of the master link sheet (Study, Outlet, Coverage URL, Study URL, Date, Source) so it's ready to paste directly.

Muckrack alerts are pulled automatically from Gmail every hour and appear in the Muckrack tabs. Any links already found by Ahrefs are automatically excluded, so there's no duplication.
